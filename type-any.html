FSharpPlus

Any

This is a wrapper type for booleans, with a specific set of monoidal operations. The contained bools would become true only if one of (any) operands are true.

Related Types

  • All: Similar wrapper, but using the 'all' criteria.

open FSharpPlus
open FSharpPlus.Data

let res1 = Any true ++ zero ++ Any false
// val res1 : Any = Any true

let even x = x % 2 = 0

let res2 = [2;4;6;7;8] |> map (even >> Any) |> sum
// val res2 : Any = Any true
